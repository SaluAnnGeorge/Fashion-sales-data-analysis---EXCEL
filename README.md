
# 📊 Fashion Retail Sales Analysis|Data Analyst Portfolio Project - Using  EXCEL

## 🔍 Project Summary

This project focuses on analyzing a **Fashion Retail Sales dataset** to uncover actionable business insights.
Using Microsoft Excel, I performed data cleaning, transformation, and visualization to support data-driven decision-making.

---

## 🎯 Business Problem

Retail businesses often struggle to:

* Identify top-performing products
* Track sales trends effectively
* Understand revenue distribution

This project aims to solve these challenges through structured data analysis.

---

## 📂 Dataset Details

* File: `Fashion_Retail_Sales final.xlsx`
* Data includes:

  * Product categories
  * Sales transactions
  * Revenue details
  * Time-based sales data

---

## 🧹 Data Cleaning & Preparation

* Removed missing and inconsistent values
* Standardized column formats
* Verified data accuracy
* Structured dataset for analysis

---

## 📊 Analysis Performed

* Sales trend analysis (time-based)
* Product category performance
* Revenue distribution
* Identification of high and low-performing products

---

## 📈 Key Insights

* 📌 Identified top-selling product categories contributing the highest revenue
* 📌 Observed seasonal trends in sales
* 📌 Highlighted underperforming products needing improvement
* 📌 Found patterns that can support inventory planning
